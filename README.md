## Getting started

Install dependencies:

``` bash
$ cd vdsite
$ npm install
```

Generate:

``` bash
$ gulp
$ hexo generate
```

Run server:

``` bash
$ hexo server
```

## Requirements

- [Hexo](http://hexo.io/)

	``` bash
  $ npm install hexo -g
  ```

- [Gulp](http://gulpjs.com/)

	``` bash
	$ npm install gulp -g
	```
